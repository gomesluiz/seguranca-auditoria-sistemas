# 🛡️ Segurança e Auditoria de Sistemas

Este espaço foi criado para centralizar os artefatos, templates e estudos de caso utilizados sobre o assunto. O objetivo aqui é tirar a segurança do campo puramente teórico e aplicar ferramentas e documentações que são padrão na indústria e no dia a dia dos profissionais área.

## 🚀 Como utilizar os templates

Os templates disponibilizados aqui não são apenas para fins acadêmicos. Eles foram desenhados para serem copiados e utilizados no seu ambiente profissional amanhã mesmo.

**Para estudantes:**
1. Faça um *Fork* ou o download deste repositório.
2. Utilize os templates na pasta correspondente para documentar a entrega dos seus estudos de caso.
3. Submeta o link do seu repositório (ou o PDF gerado) no nosso portal acadêmico.

**Para uso profissional:**
Você pode copiar o conteúdo dos arquivos `.md` e colar diretamente no Confluence, Notion, Jira ou qualquer wiki interna da sua empresa para padronizar a documentação de segurança das suas *Sprints*.
